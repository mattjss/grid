# Grid

26×26 canvas tick marks, each a damped angular pendulum tracking your cursor. Nearby marks react quickly with high amplitude; distant ones lag — creating a rolling wave of orientation that follows every move. Click fires a staggered rotational impulse that radiates outward from the impact point, temporarily overriding the cursor pull.

**Interaction:** Move cursor to orient the field · Click to fire a rotational shockwave  
**Tech:** Canvas 2D, angular spring physics, proximity-based amplitude
